# GUÍA DE PUESTA EN MARCHA PARA EL EQUIPO

¡Hola! Si acabas de descargar este proyecto, sigue estos pasos para que todo funcione correctamente, especialmente el acceso de ADMINISTRADOR.

## 1. Levantar la infraestructura
Primero, asegúrate de tener Docker corriendo y levanta los contenedores:
```bash
docker-compose up -d
```

## 2. IMPORTANTE: Sincronizar Base de Datos y Cifrado
Este proyecto utiliza seguridad profesional (Bcrypt) para las contraseñas y un sistema de roles (Admin/Estudiante). Para que tu base de datos local tenga estos datos configurados, DEBES ejecutar el script importador:

```bash
python importador.py
```
**¿Qué hace este script?**
- Crea las tablas necesarias en tu base de datos local.
- Limpia los usuarios antiguos y deja solo los autorizados (@academiametodos.com).
- CIFRA las contraseñas para que el login funcione.
- Asigna automáticamente el ROL de "admin" a los usuarios de la academia.
